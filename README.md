# コンピュータ囲碁ライブラリ「棋理」
「棋理」はRustで書かれたオープンソースのコンピュータ囲碁ライブラリです。

## 現状
碁盤モデルだけです。

### ベンチマーク
19路盤でのランダムロールアウトの速度は、

```

cargo bench -p go_rule
test tests::bench_rollout ... bench:   6,910,624 ns/iter (+/- 3,223,550)
```
